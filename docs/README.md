# Pathway Chatbot Documentation

This repository contains the source code for the Pathway Chatbot, a web-based chatbot application.

## Overview

The Pathway Chatbot is a full-stack application with a React-based frontend and a Python-based backend. The backend leverages the `llama-index` library to create a chat engine with Retrieval-Augmented Generation (RAG) capabilities.

This documentation is divided into three main sections:

- **[Project-Level Documentation](./):** High-level information about the project, including the overall architecture.
- **[Backend Documentation](./backend/docs/README.md):** Detailed documentation for the backend, including the API and setup instructions.
- **[Frontend Documentation](./frontend/docs/README.md):** Detailed documentation for the frontend, including component descriptions and setup instructions.
